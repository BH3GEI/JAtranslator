# 多语言翻译器 / Multilingual Translator

一个基于 Vue 3 的多语言翻译网页应用，支持中文、英语、日语和俄语之间的互译。

## 功能特点

- 支持中文、英语、日语、俄语四种语言的互译
- 显示详细的发音信息（包括原文读音、罗马音和国际音标）
- 响应式设计，支持各种设备

## 开发设置

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```


## 技术栈

- Vue 3
- Element Plus
- Vite
- Axios
